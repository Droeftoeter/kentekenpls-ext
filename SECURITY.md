# Security Policy

## Supported Versions

Supported versions will receive patches for security vulnerabilities

| Version | Supported          |
| ------- | ------------------ |
| 1.3.x   | :white_check_mark: |
| 1.2.x   | :x:                |
| 1.1.x   | :x:                |
| 1.0.x   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities by opening an issue on this repository.
